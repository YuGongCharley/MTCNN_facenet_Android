# charleyszc-faceDemo

---
# 简介
人脸检测算法使用mtcnn，人脸识别算法使用mobilefacenet。<br>
前向卷积框架使用ncnn。<br>
包括人脸检测、人脸对齐、人脸识别。<br>
增加活体检测。<br>

人脸对齐参考：https://github.com/WilliamEricCheung/FaceDetector/tree/fb0b6354615106895b464b7d4c32ff95528f02a8

# 注意事项
